# 插件分发说明

## 方式1：从 GitHub Actions 下载（推荐）

### 步骤：
1. 访问 GitHub 仓库：https://github.com/Genlei-Wang/HR_Agent_Boss
2. 点击 **Actions** 标签
3. 找到最新的 **Build on Push** 工作流（绿色✓表示成功）
4. 点击进入详情页
5. 滚动到页面底部，找到 **Artifacts** 区域
6. 下载 **dist** artifact
7. 解压下载的文件
8. 在 Chrome 中加载解压后的 `dist` 目录

### 优点：
- ✅ 自动构建，确保是最新代码
- ✅ 已移除所有调试代码
- ✅ 有版本记录，便于追踪

---

## 方式2：直接发送 dist 目录（快速测试）

### 步骤：
1. 压缩 `boss-greeting-assistant/dist` 目录
2. 发送给同事
3. 同事解压后，在 Chrome 中加载 `dist` 目录

### 注意事项：
- ⚠️ 确保是生产构建（运行了 `npm run build:prod`）
- ⚠️ 确保已移除调试代码

---

## 安装步骤（给同事）

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. **重要**：选择 `dist` 目录（不是 `boss-greeting-assistant` 目录）
