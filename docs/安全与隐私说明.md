# Boss招聘智能助手 - 安全与隐私说明

> 🔒 本文档说明插件的安全措施和隐私保护机制

---

## 🔐 数据安全

### API Key安全

1. **本地存储**
   - API Key仅存储在用户本地浏览器的 `chrome.storage.local` 中
   - **不会**上传到任何服务器
   - **不会**发送给第三方
   - **不会**出现在代码中（硬编码）

2. **传输安全**
   - API Key仅用于直接调用Gemini API
   - 通过HTTPS加密传输
   - 不会经过任何中间服务器

3. **访问控制**
   - 只有用户本人可以访问自己的API Key
   - 插件无法访问其他用户的API Key
   - 卸载插件时，数据会被清除

### 数据隐私

1. **候选人信息**
   - 候选人截图和分析结果仅存储在本地
   - **不会**上传到任何服务器
   - **不会**发送给第三方
   - 用户可以随时删除本地数据

2. **运行日志**
   - 所有日志仅存储在本地浏览器中
   - 不会上传到服务器
   - 用户可以导出或删除日志

3. **配置信息**
   - 职位描述、参数配置等仅存储在本地
   - 不会上传到服务器

---

## 🛡️ 安全措施

### 代码安全

1. **无硬编码密钥**
   - 代码中不包含任何API Key
   - 所有敏感信息由用户配置

2. **最小权限原则**
   - 插件仅申请必要的权限
   - 仅访问Boss直聘网站
   - 不访问其他网站数据

3. **生产构建**
   - 生产版本移除所有调试日志
   - 移除调试服务器连接
   - 优化代码，移除敏感信息

### 传输安全

1. **HTTPS加密**
   - 所有API调用使用HTTPS
   - 确保数据传输安全

2. **直接调用**
   - API调用直接发送到Gemini服务器
   - 不经过任何中间服务器
   - 减少数据泄露风险

## 📋 权限说明

### 申请的权限

| 权限 | 用途 | 必要性 |
|------|------|--------|
| `activeTab` | 访问当前活动标签页 | ✅ 必需 |
| `scripting` | 注入脚本到Boss直聘页面 | ✅ 必需 |
| `storage` | 保存配置和日志 | ✅ 必需 |
| `sidePanel` | 显示侧边栏界面 | ✅ 必需 |
| `tabs` | 查询和管理标签页 | ✅ 必需 |
| `tabCapture` | 截图功能 | ✅ 必需 |
| `downloads` | 导出日志文件 | ✅ 必需 |
| `host_permissions` | 访问Boss直聘网站 | ✅ 必需 |

### 不申请的权限

- ❌ 访问所有网站（仅访问Boss直聘）
- ❌ 读取浏览历史
- ❌ 读取书签
- ❌ 修改其他网站内容

## 🚨 安全建议

### 给HR的建议

1. **保护API Key**
   - 不要分享API Key给他人
   - 定期更换API Key
   - 如果怀疑泄露，立即更换

2. **使用公司统一API Key**
   - 建议使用公司统一的API Key服务
   - 便于管理和监控使用量

3. **定期清理数据**
   - 定期导出并删除本地日志
   - 保护候选人隐私

### 给IT部门的建议

1. **统一管理API Key**
   - 使用企业API Key服务
   - 监控API使用量
   - 设置使用限制

2. **部署安全**
   - 使用企业策略部署
   - 统一管理插件版本
   - 定期更新

3. **数据备份**
   - 定期备份重要日志
   - 确保数据安全
