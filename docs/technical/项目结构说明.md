## 🗂️ 代码目录结构说明

### `boss-greeting-assistant/` - Chrome扩展主目录

#### 核心目录

**`src/`** - 源代码目录
- **`sidepanel/`** - 侧边栏UI（React + TypeScript）
  - 使用React构建用户界面
  - 使用Zustand进行状态管理
  - 使用Tailwind CSS进行样式设计

- **`background/`** - 后台服务（Service Worker）
  - 处理API调用（Gemini API）
  - 处理截图服务
  - 处理消息传递

- **`content/`** - 内容脚本（注入到网页中）
  - 自动化操作（点击、滚动等）
  - DOM操作和候选人提取
  - 风险控制和验证码检测

- **`shared/`** - 共享代码
  - 类型定义、常量、工具函数
  - 消息类型、AI提示词

#### 配置文件

- **`package.json`** - 项目依赖和脚本
- **`tsconfig.json`** - TypeScript编译配置
- **`vite.config.ts`** - Vite构建配置
- **`tailwind.config.js`** - Tailwind CSS配置

#### 构建输出

- **`dist/`** - 构建后的文件（用于Chrome扩展加载）
  - 不要提交到Git
  - 通过 `npm run build` 生成
