# HR下载指南 - GitHub版本

> 📥 如何从GitHub下载最新版本的插件

---

## 🎯 快速下载

### 方法一：直接下载（最简单）

1. **访问下载页面**
   - 打开：`https://github.com/Genlei-Wang/HR_Agent_Boss/releases`
   - 或点击仓库首页的"Releases"链接

2. **下载最新版本**
   - 找到最新版本（通常在最上面）
   - 点击 `boss-greeting-assistant-v1.0.0.zip` 下载

3. **安装插件**
   - 按照 [HR安装使用指南.md](./HR安装使用指南.md) 的步骤安装

### 方法二：通过GitHub Desktop（如果有）

1. 克隆仓库
2. 切换到最新版本标签
3. 构建并安装

## 📋 版本说明

### 如何知道有新版本？

1. **查看Release页面**
   - 访问Releases页面
   - 查看最新版本号和发布日期

2. **查看更新日志**
   - 每个Release都有更新说明
   - 查看 [CHANGELOG.md](../../CHANGELOG.md) 了解详细变更

### 版本号说明

- `v1.0.0` - 主版本号.次版本号.修订号
- 主版本号变化：重大更新，可能不兼容
- 次版本号变化：新功能，向下兼容
- 修订号变化：Bug修复，向下兼容

## 🔄 更新插件

### 更新步骤

1. **下载新版本**
   - 从GitHub Releases下载最新ZIP文件

2. **卸载旧版本**
   - 打开 `chrome://extensions/`
   - 找到"Boss招聘智能助手"
   - 点击"移除"

3. **安装新版本**
   - 解压新下载的ZIP文件
   - 按照安装步骤重新安装

4. **配置保留**
   - 配置会自动保留（如果使用相同浏览器）
   - 如果丢失，重新配置即可

## ❓ 常见问题

### Q: 无法访问GitHub？

**解决方案：**
1. 检查网络连接
2. 尝试使用VPN
3. 联系IT部门获取代理设置
4. 或直接从项目负责人处获取ZIP文件

### Q: 下载的文件损坏？

**解决方案：**
1. 重新下载
2. 检查文件大小是否完整
3. 尝试使用不同浏览器下载

### Q: 如何知道有新版本？

**解决方案：**
1. 定期访问Releases页面
2. 关注项目更新通知
3. 查看CHANGELOG.md

### Q: 更新后配置丢失？

**解决方案：**
- 配置会自动保留（如果使用相同浏览器）
- 如果丢失，重新配置即可
- 建议记录API Key，方便重新配置
