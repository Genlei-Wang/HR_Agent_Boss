# Boss招聘智能助手 - HR安装使用指南

> 📖 本指南专为非技术人员编写，步骤详细，配有截图说明

---

## 📦 方式一：直接安装（推荐，最简单）

### 第一步：获取插件文件

1. 从IT部门或项目负责人处获取插件压缩包（`boss-greeting-assistant-v1.0.0.zip`）
2. 解压到任意位置（如桌面），会得到一个 `dist` 文件夹

### 第二步：安装到Chrome浏览器

1. **打开Chrome浏览器**

2. **进入扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或者：点击右上角三个点 → 更多工具 → 扩展程序

3. **开启开发者模式**
   - 在页面右上角，找到"开发者模式"开关
   - 点击开关，使其变为**蓝色（开启状态）**

4. **加载插件**
   - 点击左上角的"加载已解压的扩展程序"按钮
   - 在弹出的文件选择窗口中，选择刚才解压的 `dist` 文件夹
   - 点击"选择文件夹"

5. **确认安装成功**
   - 插件会出现在扩展列表中
   - 插件名称：**Boss招聘智能助手**
   - 如果看到插件图标，说明安装成功 ✅

### 第三步：固定插件图标（方便使用）

1. 点击浏览器右上角的**扩展程序图标**（拼图图标）
2. 找到"Boss招聘智能助手"
3. 点击右侧的**图钉图标**，固定到工具栏

## ⚙️ 首次配置

### 1. 获取API Key

**重要**：插件需要AI服务才能工作，需要先配置API Key。

#### 选项A：使用Gemini API（推荐，免费）

1. 访问：https://aistudio.google.com/app/apikey
2. 使用Google账号登录
3. 点击"创建API密钥"
4. 复制生成的API Key（类似：`AIzaSy...`）

**注意**：Gemini免费版每天限制20次请求，如需更多配额请升级。

#### 选项B：使用其他AI服务

请联系IT部门获取公司统一的API Key。

### 2. 配置插件

1. **打开Boss直聘网站**
   - 访问：https://www.zhipin.com/web/geek/recommend
   - 登录你的Boss直聘账号

2. **打开插件侧边栏**
   - 点击浏览器工具栏上的插件图标
   - 或点击Boss直聘页面右侧的插件图标

3. **配置API Key**
   - 在"API Key配置"区域
   - 粘贴刚才复制的API Key
   - 点击"测试连接"按钮
   - 看到"API Key验证成功"提示后，点击"保存"

4. **配置职位描述（JD）**
   - 在"职位描述JD"文本框中输入你的招聘要求
   - 例如：
     ```
     候选人-工作经历，要和B端、SaaS、工具产品、AI产品相关；
     不要和纯to C产品、垂直医疗、垂直教育、大客户定制产品相关。
     ```
   - 建议至少20字，描述越详细，匹配越准确

5. **设置参数**
   - **循环候选人数量**：建议设置为 20-50（每天）
   - **操作间隔**：建议设置为 3-8 秒（随机延迟，降低封号风险）

## 🚀 开始使用

### 使用步骤

1. **打开Boss直聘候选人列表页**
   - 访问：https://www.zhipin.com/web/geek/recommend
   - 确保页面已加载候选人列表

2. **打开插件侧边栏**
   - 点击浏览器工具栏上的插件图标

3. **检查配置**
   - 确认API Key已配置
   - 确认职位描述已填写
   - 确认候选人数量已设置

4. **点击"开始打招呼"**
   - 插件会自动开始处理候选人
   - 状态会显示为"运行中"
   - 可以看到实时进度和统计信息

5. **等待完成**
   - 插件会自动：
     - 打开每个候选人详情页
     - 截图分析工作经历
     - 判断是否匹配JD要求
     - 对匹配的候选人自动打招呼
   - 完成后会显示统计信息

6. **查看结果**
   - 在"运行日志"区域可以看到详细的操作记录
   - 匹配的候选人会标记为"已打招呼"
   - 不匹配的候选人会标记为"跳过"

### 停止运行

- 如果中途需要停止，点击"停止"按钮即可

## 📊 功能说明

### 主要功能

1. **AI智能匹配**
   - 自动分析候选人工作经历
   - 判断是否符合JD要求
   - 给出匹配度和匹配原因

2. **自动打招呼**
   - 对匹配的候选人自动发送打招呼消息
   - 跳过不匹配的候选人

3. **运行日志**
   - 记录每次操作
   - 显示匹配结果和原因
   - 支持导出日志

4. **历史记录**
   - 保存历史运行记录
   - 可以查看之前的统计信息

### 参数说明

- **循环候选人数量**：每次运行处理多少个候选人（建议不超过100人/天）
- **操作间隔**：每次操作之间的延迟时间（随机，降低封号风险）
- **职位描述JD**：你的招聘要求，AI会根据这个判断候选人是否匹配

## ⚠️ 注意事项

### 安全建议

1. **合理使用频率**
   - 建议每天不超过100次打招呼
   - 设置合理的操作间隔（3-8秒）
   - 避免短时间内大量操作

2. **API配额限制**
   - Gemini免费版每天限制20次请求
   - 如果超出限制，需要等待第二天或升级付费计划

3. **遵守平台规则**
   - 请遵守Boss直聘的使用条款
   - 合理使用自动化工具
   - 不要用于恶意刷量

### 常见问题

**Q: 插件无法启动？**
- 检查是否在Boss直聘候选人列表页
- 检查API Key是否正确配置
- 尝试刷新页面（F5）后重试

**Q: 提示"Content Script未加载"？**
- 刷新Boss直聘页面（F5）
- 重新打开插件侧边栏

**Q: API Key验证失败？**
- 检查API Key是否正确复制（不要有多余空格）
- 检查网络连接是否正常
- 确认API Key是否有效（未过期）

**Q: 截图不准确？**
- 确保浏览器缩放比例为100%（Ctrl+0）
- 确保页面已完全加载
- 如果问题持续，请联系技术支持

**Q: 配置丢失？**
- 配置会自动保存，刷新页面不会丢失
- 如果配置丢失，可能是浏览器清理了数据
- 重新配置即可
