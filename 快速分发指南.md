## ✅ 第一步：构建生产版本

```bash
cd boss-greeting-assistant
npm install  # 如果还没安装依赖
npm run package:prod
```

**这会自动完成：**
- ✅ 构建生产版本（移除所有调试代码）
- ✅ 清理敏感信息（API Key等）
- ✅ 打包成ZIP文件

**生成的文件：**
- `boss-greeting-assistant-v1.0.0.zip` - **直接发给HR**
- `dist-package/` - 解压后的文件夹

## 📦 第二步：分发给HR

### 方式A：发送ZIP文件（推荐）

1. 将 `boss-greeting-assistant-v1.0.0.zip` 发送给HR

1. 将 `dist-package/` 文件夹压缩后发送
2. HR解压后直接加载

## 🎯 第三步：HR安装（他们自己操作）

HR只需3步：

1. **解压文件**（如果收到ZIP）
2. **打开Chrome** → 地址栏输入 `chrome://extensions/`
3. **加载插件**：
   - 开启右上角"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择解压后的文件夹

## 📞 如果遇到问题

1. **构建失败？**
