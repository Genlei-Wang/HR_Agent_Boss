# 更新日志

所有重要的变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

---

## [1.0.0] - 2025-01-29

### ✨ 新增功能
- AI智能匹配：使用Gemini API分析候选人工作经历
- 自动打招呼：匹配成功后自动发送打招呼消息
- 运行日志：详细记录每次运行的操作日志和统计信息
- 历史记录：保存历史运行记录，支持查看和导出
- 配置持久化：API Key、参数配置自动保存，刷新后保留

### 🎨 界面优化
- 主按钮优化：只显示"开始打招呼"和"停止"
- 参数配置支持删除到空，显示"请输入"占位符
- Toast提示优化：移除启动时的提示，只保留重要提示

### 🔐 安全改进
- 移除所有调试代码
- API Key仅存储在本地，不会泄露
- 生产构建自动清理敏感信息

### 🛠️ 技术改进
- 适配不同设备和分辨率
- 支持不同网页缩放比例
- 优化截图功能，精确定位Canvas区域

### 📚 文档
- 添加HR安装使用指南
- 添加IT部署指南
- 添加安全与隐私说明
- 添加生产构建指南
